My Flix app

This web application provides users with access to information about different movies, directors, and genres.

Users can sign up, update their personal information, and create a list of their favorite movies.

My Flix app Documentation

Get data about a genre by genre's name /movies/genres/:genreName GET none A JSON object holding data about a genre. Example: { "Name": "Science Fiction", "Description": "Science fiction (or sci-fi) is a film genre that uses speculative, fictional science-based depictions of phenomena that are not fully accepted by mainstream science, such as..." }
Get data about a director by director's name /movies/directors/:DirectorName GET none A JSON object holding data about a director. Example: { "Name": "Stanley Kubrick", "Bio": "Stanley Kubrick was an American film director, producer, screenwriter, and photographer.", "Birth": "1928-01-01T00:00:00.000Z", "Death": "1999-01-01T00:00:00.000Z" }
Add a new user /users POST A JSON object containing data about the user to add (It should include "Name", "Password", and "Email"), structured like: { "UserName": "John", "Password": "1234", "Email": "abc@xyz.com", "Birthday": "2000-01-01" } User-entered JSON object and ID number
Get a list of ALL users /users GET none A JSON object holding data about all the users
Get user's info by name /users/:UserName GET none A JSON object holding data about the user. Example: { "_id": "64e9153a5df6228dd33d54dc", "UserName": "Kate", "Password": "kate0825", "Email": "kate0825@abc.com", "Birthday": "1980-08-25T00:00:00.000Z", "FavoriteMovies": [ "64e80dca5df6228dd33d54cf", "64e8106d5df6228dd33d54d0" ] }
Update user info /users/:UserName PUT A JSON object containing user info that shold be changed. It should contain one of the following key/value pairs: { "UserName": String, "Email": String, "Password": String, "Birthday": data } An updated JSON object
Add a movie to user's favorites list /users/:UserName/movies/:MovieID POST none An updated JSON object including new favorite movie
Remove a movie from user's favorites list /users/:UserName/movies/:MovieID DELETE none An updated JSON object
Delete user /users/:UserName DELETE none A text message indicating the user name has been deleted
