My Flix app

This web application provides users with access to information about different movies, directors, and genres.

Users can sign up, update their personal information, and create a list of their favorite movies.

My Flix app Documentation

Request URL HTTP Method Request body data format Response body data format
Get a list of ALL movies /movies GET none A JSON object holding data about all the movies
Get data about a single movie by title /movies/:Title GET none A JSON object holding data about a single movie, containing genre, director, image URL. Example: { "Title": "2001: A Space Odyssey", "Director": { "Name": "Stanley Kubrick", "Birth_Year": 1928, "Death_Year": 1999 }, "Genre": { "Name": "Sciece Fiction", "Description": "Science fiction (or sci-fi) is a film genre that uses speculative, fictional..." }, "Release_Year": 1968, "Image_Url": "https://upload.wikimedia.org/wikipedia/en/1/11/2001_A_Space_Odyssey_%281968%29.png" }
Get data about a genre by genre's name /movies/genre/:genreName GET none A JSON object holding data about a genre. Example: { "Genre": { "Name": "Sciece Fiction", "Description": "Science fiction (or sci-fi) is a film genre that uses speculative, fictional..." }, }
Get data about a director by director's name /movies/director/:directorName GET none A JSON object holding data about a director. Example: { "Name": "Stanley Kubrick", "Birth_Year": 1928, "Death_Year": 1999 }
Add a new user /users POST A JSON object containing data about the user to add, structured like: { "Name": "John", "Email": "abc@xyz.com", "Password": "1234", "Birthday": "2000-01-01" } A text message indicating the user has been added
Update user info /users/:id PUT A JSON object containing user info that shold be changed. It should contain one of the following key/value pairs: { "Name": String, "Email": String, "Password": String, "Birthday": data } A text message indicating the user's info was updated
Add a movie to user's list of favorites users/:id/movieTitle POST none A text message indicating the movie has been added
Remove a movie from user's list of favorites /users/:id/:movieTitle DELETE none A text message indicating the movie has been removed
Delete user /users/:id DELETE none A text message indicating the user ID has been removed
